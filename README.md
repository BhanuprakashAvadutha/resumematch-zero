# ResuMatch Zero

ATS Resume Scanner - Analyze your resume against job descriptions.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## Deploy on Vercel

Push to GitHub and import to Vercel. Add the environment variables in Vercel dashboard.
